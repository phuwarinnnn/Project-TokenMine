# วิธีใช้งาน - TokenMine
อันดับแรกทำแพทช์ php ก่อน ( ลง xampp ก่อนนะจ้ะ php ver 5.6 )
วิธีทำแพทช์ก็ทำตั้งแต่ภาพที่ 1 ไปเลย

แล้วกด OK หลังจากนั้นเปิด CMD ขึ้นมา แล้วเขียนว่า php เพื่อทดสอบว่า
มันทำงานหรือไม่ ถ้าพิมพ์ php แล้วกด Enter แล้วไม่มีอะไรเกิดขึ้นแปลว่า ผ่านนนนนนนนนนนน

หลังจากที่ทำ แพทช์ php เรียบร้อยแล้ว
ให้มาที่ไฟล์ getphonetoken.txt แล้วให้เอา token จริงที่มีเพื่อนไปใส่
แล้วกด run.bat แค่นั้นก็ไปรอ นอนรอได้เลยจนกว่า cmd จะปิดตัวลง


log.txt คือ username และ password ที่ได้จาก token นั้นๆ
token.txt คือ token ที่ขุดและสามารถใช้งานได้จริง

ไม่ต้อง proxy ใดๆทั้งสิ้น
นอกนั้นก็ประยุคเอาเอง
จะได้มากน้อยแค่ไหนอยู่ที่ token ต้นทุนที่เอาไปใส่ใน getphonetoken.txt
